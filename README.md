# ESP32_WS281x

This is an Arduino library that uses the Espressif ESP32 RMT peripheral to control Adafruit NeoPixels, FLORA RGB Smart Pixels, WS2811, WS2812, WS2812B, SK6812, etc.


Supports:
- Arduino ESP32
